# Content

This module focuses on the theory and practice of visualising urban data. Students develop skills in creating interactive maps, dashboards and visual analytics tools to communicate insights from urban spatial data to different audiences.

# Indicative topics

Topics include: principles of data visualisation; interactive web mapping; dashboard design; D3.js and deck.gl; spatial statistics for visualisation; urban data sources; user experience design.
