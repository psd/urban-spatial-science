# Content

This module advances students' quantitative skills by applying machine learning and data science methods to spatial systems. Building on statistical foundations from CASA0007, students develop proficiency in applying computational methods to urban and spatial research questions.

# Indicative topics

Topics include: supervised and unsupervised machine learning; classification and regression; dimensionality reduction; spatial clustering; neural networks; model evaluation and validation.
