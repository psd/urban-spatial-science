# Content

This module introduces computational simulation techniques for studying urban systems. Students develop and apply agent-based models, cellular automata and other simulation tools to explore the dynamics of cities and spatial processes.

# Indicative topics

Topics include: agent-based modelling; cellular automata; network models; spatial dynamics; calibration and validation of urban simulation models; Python programming for simulation.
