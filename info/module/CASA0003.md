# Content

A group-based project module in which students work collaboratively to produce digital visualisations of urban environments, covering 2D and 3D spatial visualisation using web-based technologies.

# Indicative topics

Topics include: 3D city modelling; web-based visualisation; collaborative project work; digital storytelling with spatial data.
