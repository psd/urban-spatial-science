# Content

This module provides a foundation in quantitative methods for spatial research. Students develop skills in statistical analysis, probability theory and regression modelling applied to urban and spatial data.

# Indicative topics

Topics include: descriptive statistics; probability and distributions; hypothesis testing; linear and logistic regression; spatial statistics; Python for data analysis.
