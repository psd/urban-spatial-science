# Urban Spatial Science MSc

Notes on the [Urban Spacial Science MSc](https://www.ucl.ac.uk/prospective-students/graduate/taught-degrees/urban-spatial-science-msc)
at [The Centre for Advanced Spatial Analysis (CASA)](https://www.ucl.ac.uk/bartlett/casa), 
a part of the 
[Bartlett Faculty of the Built Environment](https://www.ucl.ac.uk/bartlett/) 
at [University College London (UCL)](https://www.ucl.ac.uk/).
